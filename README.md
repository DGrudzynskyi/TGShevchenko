# TGShevchenko

Мені тринадцятий минало.
Я пас ягнята за селом.
Чи то так сонечко сіяло,
Чи так мені чого було?
І хлинули сльози,
Тяжкі сльози!.. А дівчина
При самій дорозі
Недалеко коло мене
Плоскінь вибирала,
Та й почула, що я плачу.
І поцілувала...
Неначе сонце засіяло,Неначе все на світі стало
Моє... лани, гаї, сади!..
