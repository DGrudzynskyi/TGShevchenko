# TGShevchenko

Мені тринадцятий минало.
Я пас ягнята за селом.
Чи то так сонечко сіяло,
Чи так мені чого було?
Прийшла, привітала,
Утирала мої сльози
Тойді так приязно молилось,
Чого так весело було.
Недовго молилось...
Запекло, почервоніло
При самій дорозі
Недалеко коло мене
Бридня!.. А й досі, як згадаю,
То серце плаче та болить,
